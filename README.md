Bodhi
Bodhi is an AI-powered hub designed to help users find, organize, and access the best AI tools across various categories like Engineering, Education, and Healthcare.

🚀 Features
Categorized AI Tools: Easily filter tools by sector (Student, Engineer, Doctor, Teacher, Editor).

Live Access: Quickly navigate to any AI tool's portal with a single click.

Secure Login: Keeps your workspace personalized.

Modern Interface: Built with a sleek, responsive design using Tailwind CSS.

🛠 Tech Stack
Framework: Flask (Python)

Frontend: HTML5, Tailwind CSS

Database: SQLAlchemy (SQLite)

Deployment: Render

This project is deployed on **Render**. Ensure your `Procfile` is configured as:
`web: gunicorn app:app`
